# typecho-qrcode-mod
拙劣修改了aneasystone的插件，修改了bug与二维码样式，改为隐藏二维码，鼠标悬停显示。

同时如果使用手机访问，此行隐藏。

并且增加了设置选项，可以调整二维码的前景色与背景色。

![Screenshot](https://github.com/signxer/typecho-qrcode/raw/master/Screenshot.jpg)

## Bug Fix
解决了没有使用jQuery的时候无法工作的问题。
